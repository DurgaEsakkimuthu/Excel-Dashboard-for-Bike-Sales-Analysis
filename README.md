# Excel-Dashboard-for-Bike-Sales-Analysis
This project analyzes customer data related to bike purchases and presents insights using an interactive Excel dashboard.

Tools Used:
🎯Microsoft Excel
🎯Pivot Tables
🎯Charts
🎯Slicers
🎯Data Cleaning Techniques

Features:
📌Interactive filters using slicers
📌Income analysis by gender
📌Bike purchase trends by age group
📌Commute distance analysis

Files Included:
📁Dataset
📁Excel Dashboard File
📁Project Report
